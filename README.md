# Data-Visualization-Portfolio

1. The Data is sourced from the official website of Australian Government.  
2. The Data is collected from an online form facility provided by AFP that can be filled by police as well as general population.  
3. Some factors are derived from pre-existinig parameters such as information about County from geolocation, year and month from crash date and time of the day from crash time.
4. All these changes are done using Google Sheets.
